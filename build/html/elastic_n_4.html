N^4 Elasticity Algorithm¶

Functions for finding the globally optimum solution to the shape energy problem. Call find_gamma() in order to produce the gamma curve, and find_energy in order to produce the minimum shape energy. Do not call find_gamma(find_energy()).

shapedist.elastic_global.find_energy()¶

Finds a the shape energy E such that E = integral from 0 to 1 of 1/2 * (p(t) - q(gamma(t))^2 is minimized.

Parameters

p : A 2-dimensional numpy array

Represents a curve P(t), with p[0] as t and p[1] as P(t)

q : A 2-dimensional numpy array

Represents a curve Q(t), with q[0] as t and q[1] as Q(t)

g : A 2-dimensional numpy array

represents the parameters of the output gamma, with g[0] as the domain of the output gamma function and g[1] as the candidate values for the gamma function

Returns

float, array of floats, array of ints

Returns the energy, the array of minimum energy values, and the position array in that order

shapedist.elastic_global.find_gamma()¶

Finds a discrete function gamma such that the E = integral from 0 to 1 of 1/2 * (p(t) - q(gamma(t))^2 is minimized.

Parameters

p : A 2-dimensional numpy array

Represents a curve P(t), with p[0] as t and p[1] as P(t).

q : A 2-dimensional numpy array

Represents a curve Q(t), with q[0] as t and q[1] as Q(t).

g : A 2-dimensional numpy array

Represents the parameters of the output gamma, with g[0] as the domain of the output gamma function and g[1] as the candidate values for the gamma function.

Returns

array of floats, float

The y - values of the gamma function corresponding to the domain and two curves implemented. Also returns the minimum shape energy.

shapedist.elastic_global.find_error()¶

Function that finds the error between two gamma curves for checking.

Parameters

tg : array of floats

The domain of the two gamma curves.

gammar : array of floats

The y-values of the known gamma curve.

gammat : array of floats

The y-values of gamma curve to be tested.

Returns

float

The weighted error.
