Welcome to 2dShapeDistance’s documentation!¶

This package is an implementation of the 2D shape elasticity algorithm as described by Javier Bernal, Gunay Dogan and Charles R. Hagwood in the paper Fast Dynamic Programming for Elastic Registration of Curves. It contains an n_4 algorithm, an n_2 algorithm, and a linear time algorithm.

Dependencies include: numpy, numba
